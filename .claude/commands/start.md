# Start Planner 

Create a new report that will explore the `Idea` using the specified `Report Format`

## Instructions 
- Research the `Idea` to see if anyone else is doing it. (Use subagents to research)
- Understand the current pricepoints for `Idea`
- Understand what oppportities there are for improvement on the `Idea`. 
- Understand what is needed to create this `Idea` MVP. 
- IMPORTANT: The user has literally $0 to accomplish this `Idea`. 
- Think how can we spend the least amount of time to have an MVP and validate the `Idea`
- Consider if we were using Claude Code for the full development of the `Idea` how long would that take?
